# sdl-software-renderer

SDL2 기반 CPU 소프트웨어 3D 렌더러. 렌더링 파이프라인/수학/셰이딩을 연구하면서
Windows/macOS 멀티 플랫폼 빌드를 유지한다.

## 문서
- 메모 및 연구 정리: `docs/Memo.md`
- 구현 로드맵/진행 기록: `docs/Roadmap.md`

## 로드맵 (요약)
1. Texture Mapping
2. Model Rendering
3. ASCII Rendering
4. Perspective Correction
5. Billnear Texture
6. Flat-Shading
7. Gouraude Shading
8. Half Lambert Shading
9. Phong Shading
10. Phong and BlinnPhong Shading
11. Water Shading and Fire Shading
